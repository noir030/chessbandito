# Run <picture><source srcset="https://fonts.gstatic.com/s/e/notoemoji/latest/1f6a8/512.webp" type="image/webp"><img src="https://fonts.gstatic.com/s/e/notoemoji/latest/1f6a8/512.gif" alt="🚨" width="32" height="32"></picture>
### 1. Clone the repository and navigate to the project directory:
```
git clone https://github.com/noir030/chessbandito.git
cd chessbandito
```
### 2. Install dependencies:
```
npm install
```
### 3. Start the project in development mode:
```
npm run dev
```
